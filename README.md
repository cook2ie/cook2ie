## 🧩 Overview

* 반응형 웹 구현 (Desktop / Tablet / Mobile)
* 프로젝트 중심 구조 설계
* 스크롤 기반 인터랙션 적용

* GSAP / ScrollTrigger를 활용한 스크롤 애니메이션 구현
* 프로젝트 모달 UI 구성 및 상태 제어
* 인터랙션 흐름을 고려한 DOM 구조 설계
* 반응형 대응을 위한 레이아웃 구성
